# 🎵 WPF Media Player

Это мой учебный проект. Я начал его, чтобы разобраться в C# и WPF.

## Что я изучил в процессе

- Работа с MediaElement (Play, Pause, Stop)
- Использование таймера (DispatcherTimer) для обновления UI
- События мыши (клик, перетаскивание)

- Виджет в углу экрана
